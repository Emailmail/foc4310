# foc4310 FOC 电机控制固件

基于 STM32G431CBUx 的 FOC 电机控制，支持 CAN 总线通信。

---

## CAN 通信协议 (兼容 QD4310)

**波特率: 1 Mbps** | **帧类型: 标准帧 (11-bit ID)**

### 控制帧 → 电机

地址 `0x400 + ID` (ID 范围 0x00~0x0F)，数据长度 **3 字节**:

| Byte | 说明 |
|------|------|
| 0 | 指令类型 |
| 1~2 | 控制量 |

指令类型:

| 值 | 指令 | 控制量 |
|----|------|--------|
| `0x00` | NOP | — |
| `0x01` | Enable | — |
| `0x02` | Disable | — |
| `0x03` | 电流控制 | int16 → ±10A |
| `0x04` | 速度控制 | int16 → ±1000 rpm |
| `0x05` | 角度控制 | uint16 → 0~2π rad |

### 反馈帧 ← 电机

地址 `0x500 + ID`，数据长度 **8 字节**:

| Byte | 说明 |
|------|------|
| 0 | 电机状态 (bit0 = 使能) |
| 1 | 错误码 (预留) |
| 2~3 | q轴电流 (int16 → ±10A) |
| 4~5 | 转速 (int16 → ±1000 rpm) |
| 6~7 | 角度 (uint16 → 0~2π rad) |

### 量程换算

| 物理量 | 编码 (→CAN) | 解码 (CAN→) |
|--------|------------|-------------|
| 电流 (A) | `int16 = A / 10 * 32767` | `A = int16 / 32767 * 10` |
| 转速 (rpm) | `int16 = rpm / 1000 * 32767` | `rpm = int16 / 32767 * 1000` |
| 角度 (rad) | `uint16 = rad / 2π * 65535` | `rad = uint16 / 65535 * 2π` |
